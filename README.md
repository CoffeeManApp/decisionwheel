# CoffeeMan Decision Wheel - Choice Maker mini app

A simple, cute, and gamified web app designed to help you make decisions easily and shake off choice paralysis. Just add your custom options, spin the wheel, and let fate decide! Built with HTML, CSS, and JavaScript.

## 🚀 Features

✅ Easily add and manage custom choices/items on the wheel.  
✅ Cute, gamified UI with thick playful borders and a pastel blue color theme.  
✅ Smooth spinning physics with an enthusiastic win animation.  
✅ Stored locally in your browser (automatically remembers your options for next time).  
✅ Lightweight and backend-free (no account or login required).  

## 📱 How to Use

Type an option into the input box (e.g., "Bobalove 🧋" or "Gaming 🎮").

Click the "Add" button or press the Enter key to add it to your wheel.

Need to modify your list? Click the "×" on any item tag to delete it.

Click the big "🎉 SPIN! 🎉" button to start the wheel.

Wait 4.5 seconds for the wheel to slow down and reveal your random result with a fun bounce animation!

## 🔐 Data Storage & Privacy

All data is saved locally on your device using Local Storage.
This means your custom option lists stay private and are never uploaded anywhere.
If you clear your browser cache or use another device, your lists will reset to the defaults.

## 🛠️ Tech Stack

- HTML5 Canvas for dynamic wheel rendering
- CSS Custom Properties for the playful pastel game theme
- JavaScript (Vanilla) for spin physics and win tracking
- LocalStorage for preserving user options offline

# CoffeeMan 趣味大转盘 极简游戏风迷你小程序

一个可爱又好玩的网页版趣味大转盘，帮你轻松搞定选择困难症！只需输入你的自定义选项，一键转动轮盘即可得到答案。基于 HTML、CSS 和 JavaScript 开发。

## 🚀 功能特点

✅ 自由添加和管理转盘上的自定义选项。  
✅ 趣味游戏风 UI，搭配粗线条漫画边框与清爽的浅蓝色调。  
✅ 逼真的转盘物理减速动效与专属中奖弹跳动画。  
✅ 数据自动保存在浏览器本地（下次打开依然保留你的专属列表，无需注册）。  
✅ 纯前端轻量级结构，无后端，秒开即用。  

## 📱 使用方法

在输入框中输入你的选项（例如：“喝奶茶 🧋”或“去睡觉 💤”）。

点击“Add”按钮或敲击键盘回车键，即可将选项加入转盘。

如果需要精简选项，直接点击标签上的“×”即可快速删除。

点击大大的“🎉 SPIN! 🎉”按钮，轮盘就会开始高速旋转。

等待 4.5 秒，转盘停止后就会伴随着可爱的弹跳动画为你揭晓最终答案！

## 🔐 数据存储与隐私

所有数据都保存在你设备的本地存储（Local Storage）。
你的选项列表完全保存在本地，绝对不会上传到任何服务器，安全私密。
如果你清空浏览器缓存或更换设备，选项列表会恢复为默认值。
